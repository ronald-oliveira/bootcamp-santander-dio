Introdução ao HTML

(voltar)

Em sua essência, o HTML e uma linguagem bastante simples
composta de elementos, que podem ser aplicados a pedaços de texto
para dar-lhes significado diferente em um documento (é um parágrafo?
é um título? é uma lista?,é parte de uma tabela?), estrutura um documento
em seções lógicas (Possui um cabeçalho? três colunas de conteúdo?
um menu de navegação?) e incorpora conteúdo como imagens e vídeos
em uma página.

HTML é uma linguagem de marcação, o que significa que ela usa "tags"
para definir elementos dentro do documento. Essas tags são envolvidas por
colchetes angulares, como <tag>, e geralmente vêm em pares: uma tag de abertura
como em <tag> e uma tag de fechamento como em </tag>.

Elementos HTML

(voltar)

Os elementos HTML são os blocos de construção de uma página web.
Cada elemento é definido por uma tag, que pode conter atributos para
fornecer informações adicionais sobre o elemento.

Por exemplo, o elemento "p" define um parágrafo, enquanto o elemento "h1"
define um título de nível 1. Os elementos podem ser aninhados uns dentro dos outros,
permitindo a criação de estruturas complexas.

Atributos de Elementos

(voltar)

Atributos são usados para fornecer informações adicionais sobre um elemento HTML.
Eles são especificados dentro da tag de abertura do elemento e geralmente vêm em pares
de nome e valor, como class="nome-da-classe" ou id="identificador".
Os atributos podem afetar o estilo, o comportamento e a funcionalidade do elemento.

Estruturas Básicas

Uma estrutura HTML básica inclui a declaração do tipo de documento,
a tag raiz <html>, e as seções <head> e <body>. A seção <head> contém
metadados, links para arquivos CSS e scripts JavaScript, enquanto a seção <body>
contém o conteúdo visível da página.

Links e Navegação

(voltar)

Links são criados usando a tag <a> (âncora) e o atributo href para especificar o destino do link.
Eles são essenciais para a navegação na web, permitindo que os usuários se movam entre páginas e sites.
A navegação pode ser facilitada por menus, barras de navegação e outros elementos interativos.

Por exemplo, um link para o Google seria escrito como:
Visite o Google.
Visite W3schools

Inserindo Imagens

(voltar)

Imagens são inseridas em uma página HTML usando a tag <img>, que é uma tag auto-fechante.
O atributo src é usado para especificar o caminho da imagem, enquanto o atributo alt fornece
uma descrição alternativa para a imagem, que é importante para acessibilidade.

Listas em HTML

(voltar)

Listas são usadas para organizar itens relacionados e podem ser ordenadas
(usando a tag <ol>) ou não ordenadas (usando a tag <ul>). Cada item da lista é definido
pela tag <li>. Listas são úteis para apresentar informações de forma clara e estruturada.
